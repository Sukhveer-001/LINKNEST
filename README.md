# LinkNest

A production-grade link-in-bio platform built with:

- Next.js 14
- NextAuth
- MongoDB
- Tailwind CSS

## Features

- Authentication (NextAuth)
- Public profile pages
- Dashboard management
- Secure middleware routing
- Responsive UI

## Setup

Clone the repo:

git clone https://github.com/Sukhveer-001/linknest.git

Install dependencies:

npm install

Run development server:

npm run dev

## Environment Variables

Create a `.env.local` file:

MONGODB_URI=
NEXTAUTH_SECRET=
NEXTAUTH_URL=
